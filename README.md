## Getting Started

1. Open `index.html` in a modern web browser
2. No installation or build process required - it's a pure HTML/CSS/JS application
3. Start tracking your workouts immediately!

## Usage

### Starting a Quick Workout
1. Click "Start Quick Workout" on the Start tab
2. Choose from pre-configured workout splits
3. Track your sets, reps, and weights
4. Complete sets and the rest timer will auto-start

### Building a Custom Routine
1. Navigate to the "Build" tab
2. Enter a routine name
3. Select exercises from the catalog organized by body part
4. Click "Start" to begin your custom workout

### Viewing History
1. Go to the "History" tab
2. Browse your past workouts
3. Click "Open" on any workout to view detailed information
